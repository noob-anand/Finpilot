# **App Name**: FinPilot

## Core Features:

- Dashboard: Displays key financial metrics such as total cash inflow, outflow, and net cash flow, dynamically updated via backend APIs.
- Invoice Management: Allows users to view a list of invoices, create new invoices, and track invoice status (paid/unpaid).
- Transaction Tracking: Enables viewing of inflow and outflow transactions and simulates payments to track business expenses and income.
- AI Copilot Chat: Provides a text-based conversational UI where users can ask questions about their financial data.
- AI-Powered Insights: The AI Copilot interprets financial data to provide plain English explanations and actionable recommendations, helping users understand financial health, detect cash flow issues, and suggest improvements. The Copilot will use financial summarization as a tool.

## Style Guidelines:

- Primary color: Deep blue (#293462) to convey trust and stability, inspired by traditional finance but updated with a modern feel.
- Background color: Light blue (#D8E2DC), a desaturated hue of the primary color.
- Accent color: Warm orange (#FF731D), an analogous hue of the primary to create good contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for both headlines and body text.
- Use clear and simple icons to represent financial concepts and actions, ensuring they are easily understandable.
- Minimalist design focused on clarity and ease of use, avoiding financial jargon and prioritizing insight-focused presentation.
- Subtle animations for loading data and transitioning between views to provide a smooth user experience.
